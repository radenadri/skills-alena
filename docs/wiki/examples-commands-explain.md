# /explain Command Examples

> Explain code, architecture, or concepts.

## Usage

```
@/explain [path-or-concept]
```

## Examples

```
@/explain src/auth/
@/explain "How does the caching layer work?"
@/explain src/middleware/rate-limit.ts
```

## Output

Detailed explanation with:
- Purpose
- How it works
- Key components
- Data flow
- Example usage
