# /init-project Command Examples

> Initialize project with .planning/ structure.

## Usage

```
@/init-project
```

## Creates

```
.planning/
├── MEMORY.md
├── STATE.md
├── sessions/
├── decisions/DECISIONS.md
├── plans/
├── context/
│   ├── architecture.md
│   ├── patterns.md
│   ├── gotchas.md
│   └── tech-debt.md
└── handoffs/LATEST.md
```
