# /prd Command Examples

> Claude Code PRD drafting through an interview-first flow.

## Usage

```bash
@/prd [feature-or-product-problem]
```

## Examples

```bash
@/prd "Create a PRD for usage-based billing"
@/prd "Draft a product requirements document for audit logging"
@/prd "Write a feature brief for customer-managed API keys"
```

## Output

Claude Code should respond with:
- clarifying questions before drafting
- a product-focused PRD structure
- goals, non-goals, and success criteria
- a clean handoff into planning after approval
