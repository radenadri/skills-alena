# /prd Workflow Examples

> Antigravity PRD drafting through an interview-first flow.

## Usage

```
/prd [feature-or-product-problem]
```

## Examples

```
/prd "Create a PRD for role-based access control"
/prd "Draft a product requirements document for webhook retries"
```

## Output

Antigravity responds with:
- clarification before drafting
- a product-first PRD structure
- explicit goals, non-goals, and success criteria
- a clear next move into planning once the PRD is accepted
